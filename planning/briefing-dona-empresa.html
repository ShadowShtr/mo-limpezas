3. Equipas e Operação Diária
Q11
Quando um serviço é atribuído a uma equipa, como são definidos os colaboradores?
Respondido
A equipa toda vai sempre junta — é uma unidade indivisível

Equipa é só um rótulo — gestor escolhe individualmente sempre
Q12
O que acontece quando um serviço precisa de mais pessoas do que a equipa tem?
Respondido
Juntam-se duas equipas no mesmo serviço
A equipa tem tamanho fixo — nunca muda
Gestor adiciona colaboradores avulso — equipa base + reforços específicos para esse serviço
Q13

Não — um contrato = uma equipa fixa para todos os dias

4. Agendamento e Calendário
Q19
Como preferes gerir os serviços recorrentes?
Respondido
Integrado no calendário — criar serviço e marcar repetição no mesmo passo
Secção separada de Contratos Fixos — lista de todos os contratos ativos. Calendário mostra as ocorrências geradas. Melhor para 20+ locais recorrentes.
Q20

Respondido
Só altera aquela ocorrência — para mudar o padrão, vai a Contratos Fixos
Pergunta: só esta / esta e todas as futuras (como Google Calendar)
